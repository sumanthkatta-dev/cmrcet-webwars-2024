# CMRCET Website (HTML/CSS)

## Structure
- index.html — main single-page layout
- assets/css/style.css — styling and layout
- assets/js/main.js — navigation toggle and smooth scrolling
- assets/images/ — place your images here

## How to use
1) Put the provided images into assets/images/ with these filenames:
   - logo: logo.png
   - main building: campus-building.jpg
   - aerial view: campus-aerial.jpg
   - library interior: library.jpg
2) Open index.html in the browser.

## Notes
- Responsive layout with sticky top navigation and mobile menu.
- Colors: deep ink, warm orange accent, teal secondary, sand background for contrast.
- Fonts: Manrope (UI) and Playfair Display (headings) from Google Fonts.
- Sections: Home, About, Academics/Departments, Admissions, Placements, Library, Contact.
